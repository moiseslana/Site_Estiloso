## 🚀 Site_Estiloso

Link do site: https://moiseslana.github.io/Site_Estiloso/

Um projeto de estudo desenvolvido com HTML e CSS, inspirado em um tutorial do YouTube, 
<br> com foco em aprender na prática como criar interfaces modernas e bem estruturadas.

## 💡 Sobre o projeto

Esse site simula um portfólio profissional, contendo seções como apresentação, serviços e trabalhos.
<br> A ideia aqui não foi só copiar, mas entender como cada parte funciona e desde o layout até os efeitos visuais.

## 🧠 O que eu pratiquei:
- Estruturação de páginas com HTML
- Estilização com CSS
- Uso de Flexbox e Grid
- Criação de layout moderno
- Aplicação de efeitos visuais (gradiente, blur, etc.)

## 🛠️ Tecnologias
- HTML5
- CSS3
